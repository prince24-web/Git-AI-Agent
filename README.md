# 🧑‍💻 Git Agent

An AI-powered agent that executes Git commands using LangChain + Gemini 2.0.

## 🚀 Features
- Initialize repos
- Clone repos
- Commit, push, pull
- Branch management
- Reset, merge, logs

## ⚡ Setup

```bash
git clone https://github.com/<YOUR_USERNAME>/<REPO_NAME>.git
cd <REPO_NAME>
pip install -r requirements.txt
